Máquinas virtuales efímeras
Estas máquinas se caracterizan por su rápida creación, lo que las hace ideales para situaciones donde se necesita una instancia de forma eficiente. Al ser de un solo uso, ofrecen un entorno limpio, no se mantienen persistentes, lo que significa que cualquier cambio o información almacenada en ellas se perderá una vez que la máquina se apague.
Máquinas virtuales CLI
Son máquinas virtuales que se pueden acceder y controlar a través de una interfaz de línea de comandos, facilitan la gestión y el control del sistema operativo de forma directa, no poseen interfaz gráfica de usuario (GUI) para evitar la congestión de red y que tanto los usuarios en el Desktop Cloud como los usuarios directos en el host no se vean afectados.
Usuarios invitados
Los usuarios invitados pueden acceder al sistema de forma rápida y sencilla sin necesidad de registrarse, lo que les permite experimentar con la plataforma de manera temporal y sin compromisos.
